Zivara – Mental Wellness Web App

Zivara is a mental wellness web application built using Flutter Web and deployed using GitHub Pages. The app provides calming tools, emergency support, and a safe memory space to help users manage stress and emotional situations.

📱 About Zivara
Zivara is designed to support emotional well-being through simple and accessible digital tools. It includes features that help users stay calm during stressful moments and connect with trusted contacts when needed.

✨ Key Features
🚨 Crisis Mode – Quick-access tools for stressful situations

🧘 Breathing Exercises – Helps users relax and regain focus

🛟 Safe Haven – Store meaningful photos and videos

📩 SOS Messaging – Alert trusted contacts in emergencies

🎨 Calm UI – Soft purple to blue gradient wellness theme

🛠 Built With
Flutter Web

Dart

HTML/CSS (Generated via Flutter Web)

GitHub Pages (Deployment)

🚀 How This Project Was Deployed
The web version of Zivara was generated using:

flutter build web
The contents of the build/web folder were uploaded to the GitHub repository and hosted using GitHub Pages.

📌 Note
This is a demonstration version of the Zivara mental wellness application developed as an academic project.

⚠️ Disclaimer
Zivara provides wellness support tools but does not offer medical or psychological advice.
In case of emergencies, users should contact official emergency services.
